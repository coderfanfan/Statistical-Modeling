This code can be used to estimate product demand and calculate customer price sensitivity using market share data (aggregate-level instead of customer-level) with related attributes (e.g.pricing, brand) 

The sample data comes from Dominick's Database 
https://research.chicagobooth.edu/kilts/error?item=%2fkilts%2fmarketing-%C2%AD%E2%80%90databases%2fdominicks&user=ad%5cAnonymous&site=research

The folder contains:
1. code
2. zones: dataset 1
3. catsales_RefrigOJ_2017Semester: sample market share dataset
4. result: model result with estimation of price sensitivity
